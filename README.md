2️⃣ Teclado/Bateria

Um teclado musical que reproduz sons ao pressionar teclas ou clicar nos botões.

Objetivos de aprendizado:

Manipulação do DOM

Eventos de teclado e clique

Lógica para mapear sons diferentes para cada tecla

Funcionalidades:

Reproduz sons ao apertar teclas correspondentes

Feedback visual ao clicar nos botões

Interface interativa para prática de eventos e funções
